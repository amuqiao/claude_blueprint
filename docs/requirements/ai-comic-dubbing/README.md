# AI 漫剧字幕与配音替换文档

本目录围绕 AI 漫剧平台中的字幕替换、配音替换和音色相似需求，按”扫盲 → 方法论 → 选型 → 落地”的顺序组织文档。

---

## 信息流动关系

```text
  基础扫盲
    ↓ 建立对象理解
  方法论
    ↓ 定义执行流程(9节点)
  技术选型
    ↓ 为每个节点选择技术方案
  MVP 落地方案
    ↓ 给出工程实现

  这个顺序的合理性:
  1. 基础扫盲 → 方法论: 理解了问题是什么,才能定义应该怎么做
  2. 方法论 → 技术选型: 知道了需要做哪些节点,才能为每个节点选择技术
  3. 技术选型 → MVP: 选定了技术路线,才能给出具体实现
```
