# 召唤牌库超能力组与升级路线

> 召唤不是“叫出一个帮手”，而是把武器、契约体、临时异能、增益、减益和随机抽取整合成一套可管理、可锁定、可组合的战术牌库。

## 能力锚点和一句话玩法

```text
能力锚点：召唤牌库
一句话玩法：通过抽取、锁定、召唤、献祭和特殊召唤，在任务中临时获得武器、使魔、异能、Buff、Debuff 或团队支援。
核心限制：召唤结果来自牌库和契约池，越稀有、越跨规则、越长时间维持，代价越高。
```

这套能力的直觉不是“我本来会什么”，而是“我能从牌库里调出什么”。

能力者本人可以没有火焰、治疗、隐身、精神控制或空间位移异能，但可以通过召唤武器、契约体、临时技能卡、增益光环或负面诅咒，把这些能力以外接形式接入战斗和任务。它的上限来自牌库深度、召唤规则、锁定能力和临场组合，而不是来自单一召唤物的强度。

## Seed：创意种子

```text
原始输入：召唤武器、召唤魅魔、随机召唤、抽卡机制、锁定结果、特殊召唤、借用自己没有的异能、Buff 和 Debuff。
类型：召唤 / 抽卡 / 契约 / 临时构筑型超能力
吸引点：能力者可以把“没有的能力”变成“临时可调用资源”，形成高度全能但有规则约束的玩法。
容易俗套化：只写成召唤兽打架、召唤武器砍人、抽到强卡就赢。
```

这套能力要避免把召唤理解成单纯“多一个单位”。更有趣的方向是：召唤系统本身就是能力核心，抽取、保底、锁定、弃牌、献祭、连锁和特殊召唤都能变成任务解法。

## Lens：解释维度

召唤可以从多个维度解释：

```text
实体维度：召唤武器、怪物、使魔、护卫或工具
契约维度：与不同存在签订条件，换取能力、服务或代价转移
抽卡维度：能力以随机池形式出现，强调概率、保底、锁定和重抽
构筑维度：提前编辑牌库，决定任务中可能抽到什么
附魔维度：召唤不是实体，而是给自己、队友或敌人挂上状态
规则维度：通过献祭、融合、连锁、特殊召唤改变召唤条件
```

本文选定的主维度是：

```text
主维度：抽卡维度 + 契约维度 + 构筑维度
表现维度：实体召唤 + 状态召唤
```

理由是：抽卡维度提供随机性和兴奋感，契约维度提供代价与角色感，构筑维度提供策略深度。实体召唤和状态召唤作为表现层，负责让武器、魅魔、临时异能、团队 Buff 和敌方 Debuff 都能进入同一套系统。

## Operation：操作面

召唤牌库的操作面可以拆成九类：

```text
牌库：提前收录可召唤的武器、契约体、技能、状态和道具
抽取：从牌库中随机获得本次可用召唤项
锁定：把某个结果固定到手牌、装备位、队友或场景节点上
重抽：放弃当前结果，换取新的召唤机会
契约：用代价、条件或关系换取召唤物服务
附着：把召唤结果挂到自身、队友、敌人、武器或区域上
献祭：消耗召唤物、资源、状态或风险，换取更高阶召唤
连锁：让一个召唤结果触发另一个召唤或状态变化
```

这些操作面决定能力的统一性。召唤武器是“实体牌”，召唤魅魔是“契约体牌”，临时获得火焰异能是“技能牌”，给队友上护盾是“增益牌”，给敌人挂虚弱是“减益牌”，随机召唤和抽卡机制则是整套能力的资源调度方式。

## Rule：核心能力规则

```text
能力者不能无限召唤任意事物，只能从已收录、已契约或临时接入的牌库中抽取结果。
随机抽取决定可用机会，锁定和构筑决定可控程度。
召唤物越强、越稀有、越跨能力体系，越需要代价、祭品、冷却或契约条件。
同一时间可维持的召唤位有限，超出上限会导致召唤崩解、契约反噬或牌库污染。
```

这条规则让能力既全能，又不会变成无条件许愿。

例如能力者自己没有治疗异能，但如果牌库中有“白瓷护士”契约体，或者有“再生祷言”技能卡，就可以临时召唤治疗效果。问题在于：他不一定抽得到；抽到后也可能需要血液、时间、精神专注或任务目标作为代价；如果提前锁定治疗牌，就会牺牲其他输出、潜入或逃生选项。

## Type：能力类型

```text
能力本体类型：召唤 / 契约 / 抽卡构筑
主要技能类型：武器、使魔、临时异能、增益、减益、控制、探索、资源、反制
玩法定位：高灵活、高组合、高不确定性的战术构筑型能力
```

它和相邻能力的差异是：

| 相邻能力 | 差异 |
| --- | --- |
| 普通召唤兽 | 普通召唤兽强调固定伙伴，召唤牌库强调可抽取、可构筑、可连锁的资源系统 |
| 武器召唤 | 武器召唤只解决装备问题，召唤牌库还能召唤状态、契约体和临时异能 |
| 复制能力 | 复制能力依赖目标原本拥有的能力，召唤牌库依赖收录、契约和牌库构筑 |
| 幸运抽卡 | 幸运抽卡只改变结果好坏，召唤牌库能锁定、重抽、献祭和特殊召唤 |
| 亡灵召唤 | 亡灵召唤通常限定死者或尸体，召唤牌库的对象可以是武器、精灵、恶魔、概念状态或技能影像 |

## Layer：成长层级

召唤牌库的成长不是“召唤物越来越强”这么简单，而是从随机叫出物品，逐步成长为可构筑、可锁定、可连锁、可特召的完整系统。

```text
第 0 层：随机召唤
第 1 层：牌库构筑
第 2 层：锁定与重抽
第 3 层：契约体与状态附着
第 4 层：连锁与特殊召唤
第 5 层：规则级召唤
```

| 层级 | 能力含义 | 主要解锁 |
| --- | --- | --- |
| 第 0 层：随机召唤 | 从小型牌库随机获得武器、道具或低阶使魔 | 随机武装、应急工具、临时护卫 |
| 第 1 层：牌库构筑 | 任务前编辑可抽取内容，减少废牌 | 武器池、潜入池、治疗池、逃生池 |
| 第 2 层：锁定与重抽 | 固定关键结果，丢弃不适合的召唤项 | 保底召唤、手牌锁定、命运重抽 |
| 第 3 层：契约体与状态附着 | 召唤具备人格或规则的契约体，并挂载 Buff/Debuff | 魅魔契约、护卫契约、祝福、诅咒 |
| 第 4 层：连锁与特殊召唤 | 用献祭、融合、触发条件换取高阶结果 | 献祭召唤、融合召唤、陷阱触发 |
| 第 5 层：规则级召唤 | 短时间召唤一条局部规则或战场机制 | 抽卡领域、全队增益、敌方禁用 |

第 5 层不是无条件现实改写。它只是把召唤系统扩展成区域规则，例如“本区域每次攻击后抽一张状态牌”或“敌方每次发动能力前先接受一次 Debuff 判定”。

核心成长路径可以是：

```text
随机武装
-> 三选一
-> 保底锁定

-> 牌桌展开
```

这个路径表达从“抽到什么用什么”，到“挑选和锁定结果”，再到“召唤有规则的契约体”，最后把战场变成可连锁的召唤牌局。

## Power：关键能力组

| 能力 | 类型 | 层级 | 操作面 | 一句话效果 | 基础限制 |
| --- | --- | --- | --- | --- | --- |
| 随机武装 | 武器 / 主动 | 第 0 层 | 抽取 | 随机召唤一件适合当前战斗距离的武器 | 结果不稳定，可能不是最优武器 |
| 三选一 | 抽卡 / 主动 | 第 1 层 | 抽取 / 构筑 | 抽出三个召唤选项，从中选择一个发动 | 需要短暂停顿，紧急情况下容易被打断 |
| 保底锁定 | 构筑 / 被动 | 第 2 层 | 锁定 | 将一张关键牌设为保底，若多次未出现则必定进入手牌 | 锁定会占用牌库权重和召唤位 |
| 命运重抽 | 抽卡 / 主动 | 第 2 层 | 重抽 | 丢弃当前结果，重新抽取一次召唤项 | 重抽次数越多，牌库污染越高 |
| 契约兵器 | 武器 / 主动 | 第 2 层 | 契约 | 召唤带有特殊规则的武器，如破甲、静音、眩晕或封印 | 兵器有使用条件和契约代价 |
| 魅魔契约 | 使魔 / 控制 | 第 3 层 | 契约 / 附着 | 召唤擅长魅惑、误导、情报套取和精神 Debuff 的契约体 | 对高意志目标效果下降，契约体可能索取代价 |
| 影仆 | 使魔 / 潜入 | 第 3 层 | 契约 | 召唤低存在感仆从侦查、搬运、偷取钥匙或触发机关 | 害怕强光和净化类能力 |
| 勇气祝福 | Buff / 团队 | 第 3 层 | 附着 | 给自己或队友召唤一层勇气状态，提高抗恐惧、抗控制和行动稳定性 | 同类 Buff 不能无限叠加 |
| 虚弱诅咒 | Debuff / 控制 | 第 3 层 | 附着 | 给敌人挂上虚弱、迟缓、失准或易受伤状态 | 目标抗性越高，持续时间越短 |
| 借用隐身 | 临时异能 / 潜入 | 第 3 层 | 附着 | 召唤一张异能牌附着自身，短时间获得不属于自己的隐身能力 | 只能维持短时间，攻击或强光会破除 |
| 献祭召唤 | 高阶 / 主动 | 第 4 层 | 献祭 | 消耗召唤物、手牌或自身状态，换取更高阶召唤 | 祭品不足会反噬，可能失去关键资源 |
| 融合召唤 | 高阶 / 主动 | 第 4 层 | 连锁 | 将两个召唤结果融合成新实体或新状态 | 融合结果受素材兼容性影响 |
| 陷阱特召 | 反制 / 被动 | 第 4 层 | 连锁 | 当敌人触发条件时，自动召唤陷阱、护盾或反击体 | 需要提前设置触发条件 |
| 牌桌展开 | 领域 / 终极 | 第 5 层 | 规则 | 展开局部抽卡领域，让双方行动都进入抽牌、弃牌、锁定和禁卡规则 | 领域维持消耗极高，也可能让敌人利用规则 |

## Upgrade：升级路线

### 随机武装

```text
基础：随机召唤一件近战、远程或防御武器
升级 1：根据距离、敌人护甲和环境自动排除明显废牌
升级 2：可以短暂双持两件随机武装
升级 3：击败敌人后抽取一张临时武器强化牌
```

### 三选一

```text
基础：抽出三个召唤选项并选择一个
升级 1：未选择的两张牌可以转化为少量召唤能量
升级 2：可以把一张未选择牌暂存为备用手牌
升级 3：三张牌如果同类型，可触发一次免费连锁召唤
```

### 保底锁定

```text
基础：把一张关键牌设为保底
升级 1：可以为不同场景设置战斗、潜入、治疗和逃生保底
升级 2：队友濒危、自己重伤或目标逃跑时自动触发对应保底牌
升级 3：保底牌发动后不立刻消失，而是进入短暂冷却
```

### 魅魔契约

```text
基础：召唤魅魔契约体，进行诱导、误导、精神压制或情报套取
升级 1：魅魔可以伪装成目标熟悉的人影或声音
升级 2：魅魔施加的 Debuff 可以转化为己方团队 Buff
升级 3：魅魔在任务中完成契约目标后，可触发一次特殊召唤或额外抽牌
```

### 借用隐身

```text
基础：短时间获得自己原本没有的隐身能力
升级 1：隐身期间首次开门、偷取或搬动物品不破隐
升级 2：可以把隐身附着到队友、召唤物或小型物品上
升级 3：隐身结束时留下一张影仆牌，误导追踪者
```

### 献祭召唤

```text
基础：牺牲一个召唤物或一张手牌，召唤更高阶结果
升级 1：可以献祭敌方 Debuff，把敌人的弱点转化为己方资源
升级 2：可以献祭临时 Buff，换取一次爆发型特殊召唤
升级 3：多个低阶召唤物可组合献祭，稳定召唤神话武装或守门巨像
```

### 牌桌展开

```text
基础：展开局部领域，让区域内行动进入抽卡规则
升级 1：指定一类敌方能力为禁卡，短时间内无法使用
升级 2：己方每完成一次潜入、救援或非致命处理，抽一张奖励牌
升级 3：领域结束时根据剩余手牌结算终局效果，例如全队治疗、敌方缴械或安全撤离路线
```

## Loadout：能力装配示例

### 武器大师流

```text
固有位：三选一
主动位：随机武装 / 契约兵器 / 命运重抽
反制位：陷阱特召
终极位：神话武装
玩法：根据距离、敌人护甲和场景材料不断切换武器，用抽卡解决装备短板。
```

### 契约使魔流

```text
固有位：保底锁定
主动位：影仆 / 魅魔契约 / 守门巨像
状态位：虚弱诅咒
终极位：融合召唤
玩法：用影仆侦查和搬运，用魅魔套情报和挂 Debuff，用巨像封路或护卫撤离。
```

### 异能借用流

```text
固有位：三选一
主动位：借用隐身 / 临时火种 / 短距闪现
辅助位：命运重抽
终极位：牌桌展开
玩法：自己没有核心异能也能临时接入，让能力组根据任务压力快速变形。
```

### 团队增益流

```text
固有位：保底锁定
主动位：勇气祝福 / 团队光环 / 虚弱诅咒
连锁位：献祭召唤
终极位：终局抽牌
玩法：给队友稳定上 Buff，给敌人挂 Debuff，再把状态差转化为高阶召唤资源。
```

## Mission：场景验证

| 场景 | 能力组合 | 使用结果 | 代价 / 风险 |
| --- | --- | --- | --- |
| 战斗 | 随机武装 + 契约兵器 + 命运重抽 | 快速抽到适合当前距离和敌人弱点的武器 | 抽卡停顿可能被近身压制 |
| 防御 | 陷阱特召 + 守门巨像 + 勇气祝福 | 敌人突进时自动召唤防线，并稳定队友心态 | 需要提前设置触发条件 |
| 逃生 | 借用隐身 + 影仆 + 保底锁定 | 自身隐身撤离，影仆搬钥匙或触发机关，保底牌兜底 | 隐身被破后容易陷入空牌期 |
| 潜入 | 三选一 + 魅魔契约 + 虚弱诅咒 | 抽取合适工具，魅魔误导守卫，给高警觉目标挂精神 Debuff | 高意志敌人可能反向识破契约体 |
| 探索 | 影仆 + 短距闪现 + 契约兵器 | 影仆侦查窄道，闪现跨越危险点，契约兵器破除封印 | 牌库未构筑探索牌时效率下降 |
| 团队 | 团队光环 + 勇气祝福 + 终局抽牌 | 全队获得抗性、行动稳定和结算治疗 | 长时间维持会压缩个人输出牌位 |
| 控制敌人 | 魅魔契约 + 虚弱诅咒 + 献祭召唤 | 诱导敌人犯错，再把 Debuff 献祭成强控制召唤 | 对无心智目标效果有限 |
| 高阶规则 | 牌桌展开 + 禁卡宣言 + 终局抽牌 | 把战场变成抽卡领域，禁止敌方关键能力并用剩余手牌结算胜势 | 敌人若理解规则，也可能利用抽牌机制 |

## 特殊召唤规则

特殊召唤是这套能力最有趣的部分。它不是单纯变强，而是让玩家把场景、状态和代价转化成新解法。

```text
献祭召唤：消耗召唤物、Buff、Debuff、手牌或自身状态，换取更高阶召唤
融合召唤：把两个召唤结果合并，例如“火种 + 长枪”变成火焰契约枪
陷阱特召：提前设定条件，例如“敌人开门时召唤束缚链”
反向特召：敌人攻击命中时，把伤害转化为护盾或治疗召唤
契约特召：完成契约体提出的小目标后，免代价召唤对应奖励
空牌特召：手牌耗尽时触发保命召唤，但会污染牌库
```

这些规则能让召唤能力接近游戏抽卡机制，但又不只是数值抽奖。真正的玩法点在于：玩家能否用低阶牌、场景材料和敌人状态，拼出高阶召唤条件。

## Limit：边界代价

召唤牌库必须有清楚边界，否则会变成全能许愿机。

```text
不能召唤未收录、未契约或没有临时接入路径的对象
不能无限维持召唤物，同一时间召唤位有限
不能保证每次都抽到最优结果，除非提前消耗资源锁定
不能无代价借用高阶异能，越跨体系越容易反噬
不能让契约体完全无条件服从，强契约体有自身偏好和索价
不能无限叠加 Buff 或 Debuff，同类状态会衰减或冲突
不能把随机性完全删除，否则抽卡体系失去代价和张力
```

常见反噬包括：

```text
牌库污染：反复重抽、空牌特召或失败契约会让废牌、诅咒牌进入牌库
契约索价：强大召唤体要求记忆、寿命、情报、情绪或未来行动作为报酬
召唤失控：维持位不足时，召唤物提前崩解或误伤队友
规则反噬：牌桌展开后，敌人也可能获得抽牌、弃牌或触发机会
身份侵蚀：长期借用异能或契约体力量，会让能力者性格、欲望或身体状态被召唤源影响
```

常见反制方式：

```text
封印牌库：禁止抽取或切断召唤源
禁卡领域：指定某类召唤结果无法发动
契约审判：强制暴露召唤契约的真实代价
锚点破坏：摧毁召唤物维持现实存在的坐标
概率干扰：增加废牌权重，让能力者抽不到关键结果
净化光环：削弱契约体、恶魔、诅咒和精神类召唤
```

## 总结

召唤牌库的强大不在于某一个召唤物，而在于它把“没有的能力”变成“可抽取、可锁定、可献祭、可连锁的临时资源”。

低阶召唤解决“现在能抽到什么”，中阶召唤解决“我如何构筑和锁定结果”，高阶召唤解决“我如何用契约、状态和场景触发特殊召唤”。它可以非常全能，但全能来自牌库管理，而不是无条件拥有一切能力。

判断这套能力是否成立，重点看三件事：

```text
每个召唤结果是否都能解释回牌库、契约、抽取、锁定或附着机制
随机性是否制造选择，而不是让胜负只看运气
能力是否能进入具体任务解法，而不是只停留在“召唤一个很强的东西”
```
