# 瞬移超能力技能化与成长树

> 瞬移不是“随便出现在任何地方”，而是围绕定位、位移、换位、锚点、空间折叠和连续机动展开的技能成长体系。

## 能力锚点和一句话玩法

```text
能力锚点：瞬移
一句话玩法：通过消耗空间感知和位移资源，在可定位范围内完成短距闪现、锚点返回、目标换位和高阶空间折叠。
核心限制：不能瞬移到无法定位、无法承载或被空间封锁的位置。
```

瞬移的玩家直觉是“位置选择”。它的乐趣不在于单次移动距离有多远，而在于如何用有限的位移资源改变站位、躲避攻击、切入后排、救援队友、穿越障碍或撤离危险区域。

## Type：能力类型

```text
能力本体类型：空间位移 / 感知定位
主要技能类型：位移、防御、潜入、追击、救援、控制、反制
玩法定位：高机动、高操作、高风险的空间机动型能力
```

瞬移能力的核心不是速度，而是“位置关系”。它和高速移动、飞行、传送门有明确差异：

| 相邻能力 | 差异 |
| --- | --- |
| 高速移动 | 仍然沿路径移动，会被路径上的障碍、陷阱和攻击影响 |
| 飞行 | 提供持续空间优势，但不一定能跳过障碍或瞬间脱离 |
| 传送门 | 通常建立通道，可让多人或物体通过，准备和暴露更明显 |
| 瞬移 | 直接改变自身或目标位置，更强调定位、距离、冷却和承载 |

具体技能类型可以分为：

```text
自我位移：移动自身
目标位移：移动敌人、队友或物体
锚点位移：回到预设位置
换位控制：交换双方位置
残影防御：用短距位移规避伤害
空间感知：建立可瞬移的目标和坐标
空间反制：干扰、封锁或追踪其他位移
```

## Layer：成长层级

瞬移的成长不是“距离越来越远”这么简单，而是从个人短距位移，逐步进入空间定位、空间关系和局部空间规则。

```text
第 0 层：短距瞬移
第 1 层：锚点定位
第 2 层：位置交换
第 3 层：空间折叠
```

| 层级 | 能力含义 | 主要解锁 |
| --- | --- | --- |
| 第 0 层：短距瞬移 | 在可视或可感知范围内移动自身 | 闪步、相位闪避、穿隙 |
| 第 1 层：锚点定位 | 标记位置或对象，并围绕锚点移动 | 回响锚点、归位、锚点突袭 |
| 第 2 层：位置交换 | 改变两个对象的位置关系 | 换位、救援换位、错位摔投 |
| 第 3 层：空间折叠 | 暂时压缩局部空间距离 | 折距步、区域折叠、折叠牢笼 |

层级关系不是替代关系。`空间折叠` 不会删除 `闪步`，也不自动覆盖所有短距瞬移技巧。低阶技能保留为具体实现，高阶技能提供新的分支和联动方式。

## Tree：技能树

```text
瞬移技能树
├── 短距位移分支
│   ├── 闪步
│   ├── 相位闪避
│   └── 穿隙
├── 锚点分支
│   ├── 回响锚点
│   ├── 归位
│   └── 锚点突袭
├── 换位分支
│   ├── 影换
│   ├── 救援换位
│   └── 错位摔投
├── 感知分支
│   ├── 空间测距
│   ├── 落点预判
│   └── 位移追踪
├── 折叠分支
│   ├── 折距步
│   ├── 区域折叠
│   └── 折叠牢笼
└── 反制分支
    ├── 空间锁钉
    ├── 位移扰流
    └── 锚点破坏
```

技能树的核心路径可以是：

```text
闪步
-> 回响锚点
-> 影换
-> 折距步
```

这个路径表达从“移动自己”到“绑定位置”，再到“改变双方位置”，最后到“改变局部空间距离”的成长。

## Skill：关键技能表

| 技能 | 类型 | 层级 | 分支 | 描述 | 条件 | 消耗 | 冷却 | 升级 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 闪步 | 主动 / 位移 | 第 0 层 | 短距位移 | 瞬移到短距离内可感知落点 | 落点可感知且可承载身体 | 位移能量 | 短 | 增加距离，降低落点硬直 |
| 相位闪避 | 主动 / 防御 | 第 0 层 | 短距位移 | 在攻击命中前短距偏移，规避一次伤害 | 需要预判攻击方向 | 位移能量 + 反应窗口 | 中 | 成功闪避后返还部分资源 |
| 穿隙 | 主动 / 潜入 | 第 0 层 | 短距位移 | 穿过门缝、栅栏、薄墙等空间缝隙 | 目标另一侧可感知 | 位移能量 | 中 | 可携带小型物品 |
| 回响锚点 | 主动 / 标记 | 第 1 层 | 锚点 | 在当前位置留下短时空间锚点 | 当前位置稳定且未被封锁 | 精神专注 | 中 | 延长锚点持续时间 |
| 归位 | 主动 / 撤离 | 第 1 层 | 锚点 | 返回最近一个有效锚点 | 锚点未被破坏 | 位移能量 | 长 | 可清除一次负面位移状态 |
| 锚点突袭 | 主动 / 追击 | 第 1 层 | 锚点 | 瞬移到被标记目标附近 | 目标携带锚点标记 | 位移能量 | 中 | 落地后获得短时加速 |
| 影换 | 主动 / 控制 | 第 2 层 | 换位 | 与一个短距离目标交换位置 | 目标被锁定且质量差距不大 | 位移能量 + 精神负荷 | 长 | 可对召唤物或残影使用 |
| 救援换位 | 主动 / 辅助 | 第 2 层 | 换位 | 与队友换位，替其脱离危险落点 | 队友同意或处于救援标记 | 位移能量 | 长 | 换位后给队友短时护盾 |
| 错位摔投 | 主动 / 控制 | 第 2 层 | 换位 | 将敌人位置短距错开，使其失衡或撞入障碍 | 需要命中空间锁定 | 位移能量 | 中 | 增加失衡时间 |
| 空间测距 | 被动 / 感知 | 第 1 层 | 感知 | 感知可瞬移落点的距离、承载和危险程度 | 持续专注 | 精神负荷 | 无 | 提高落点判断精度 |
| 落点预判 | 被动 / 防御 | 第 1 层 | 感知 | 显示短时间内高风险落点 | 已解锁空间测距 | 精神负荷 | 无 | 扩大预判范围 |
| 位移追踪 | 被动 / 反制 | 第 2 层 | 感知 | 感知附近位移痕迹和锚点波动 | 目标刚发生空间移动 | 精神负荷 | 无 | 可短时追踪残留路径 |
| 折距步 | 主动 / 高阶位移 | 第 3 层 | 折叠 | 将两点之间距离短时压缩，完成超出常规范围的瞬移 | 两点均可定位 | 大量位移能量 | 长 | 降低空间眩晕 |
| 区域折叠 | 主动 / 领域 | 第 3 层 | 折叠 | 压缩局部区域距离，让己方移动更短、敌方判断失准 | 需要稳定空间中心 | 位移能量持续消耗 | 长 | 增加持续时间 |
| 折叠牢笼 | 主动 / 控制 | 第 3 层 | 折叠 | 让目标周围空间距离异常，限制其移动路线 | 目标处于区域折叠内 | 高精神负荷 | 长 | 降低目标位移技能成功率 |
| 空间锁钉 | 主动 / 反制 | 第 2 层 | 反制 | 短时间固定某一区域，阻止瞬移落点生成 | 需要提前布置 | 空间钉 / 能量 | 长 | 扩大封锁半径 |
| 位移扰流 | 主动 / 反制 | 第 2 层 | 反制 | 干扰敌方位移，使其落点偏移 | 目标正在发动位移 | 精神负荷 | 中 | 增加偏移距离 |
| 锚点破坏 | 主动 / 反制 | 第 2 层 | 反制 | 破坏可感知的空间锚点 | 需要发现锚点 | 位移能量 | 中 | 可吞噬部分锚点能量 |

## Loadout：技能装配示例

### 机动刺客流

```text
固有位：空间测距
主动位：闪步 / 锚点突袭 / 影换
被动位：落点预判
终极位：折距步
玩法：快速切入、换位扰乱、锚点撤离，适合单点爆发和追击。
```

### 救援游侠流

```text
固有位：空间测距
主动位：回响锚点 / 归位 / 救援换位
被动位：落点预判
装备位：空间锁钉
玩法：提前布锚、救援队友、封锁敌方追击，适合团队保护和撤退。
```

### 空间控制流

```text
固有位：位移追踪
主动位：位移扰流 / 空间锁钉 / 区域折叠
被动位：落点预判
终极位：折叠牢笼
玩法：削弱敌方机动能力，制造错误落点，适合防守和区域控制。
```

## Combo：组合玩法

| 组合 | 操作逻辑 | 结果 | 风险 |
| --- | --- | --- | --- |
| 回响锚点 -> 闪步 -> 归位 | 先设置撤退点，再切入，最后返回 | 安全试探和撤退 | 锚点被破坏会失去退路 |
| 闪步 -> 影换 -> 错位摔投 | 接近目标后换位，再制造失衡 | 打乱敌人站位 | 对重量差距大的目标效果下降 |
| 空间测距 -> 穿隙 -> 锚点突袭 | 先确认落点，再穿越障碍，最后追击标记目标 | 潜入和追杀 | 落点不可承载会失败 |
| 救援换位 -> 回响锚点 -> 归位 | 与队友换位后把自己拉回锚点 | 救援队友并减少自身风险 | 需要提前布置锚点 |
| 位移扰流 -> 空间锁钉 -> 折叠牢笼 | 干扰敌方位移，再封锁落点，最后限制移动路线 | 强控制区域 | 消耗高，准备时间长 |
| 区域折叠 -> 折距步 -> 影换 | 改变区域距离后跨距切入，再换位 | 高阶突袭 | 空间反噬风险高 |

## Scene：场景验证

| 场景 | 技能组合 | 使用结果 | 代价 / 风险 |
| --- | --- | --- | --- |
| 战斗 | 闪步 + 影换 + 错位摔投 | 快速接近并打乱敌方阵型 | 需要精确判断目标质量和落点 |
| 防御 | 相位闪避 + 归位 | 规避伤害并回到安全锚点 | 预判失败会吃满伤害 |
| 逃生 | 回响锚点 + 穿隙 + 归位 | 穿过障碍后返回预设撤离点 | 锚点被追踪可能暴露路线 |
| 潜入 | 空间测距 + 穿隙 | 绕过门锁、栅栏和巡逻路径 | 无法感知另一侧时不能发动 |
| 探索 | 空间测距 + 折距步 | 跨越断崖、深坑或迷宫距离 | 高阶折叠可能造成空间眩晕 |
| 生活 | 闪步 + 空间测距 | 快速搬运、避开拥堵、取回物品 | 频繁使用会疲劳和方向感错乱 |
| 团队 | 救援换位 + 空间锁钉 | 救出队友并阻止敌方追击 | 队友位置误判会导致双人陷险 |

## Limit：边界代价

瞬移必须有清楚边界，否则很容易变成无解能力。

```text
不能瞬移到完全无法感知或无法定位的位置
不能瞬移到无法承载身体的空间，例如实体内部、过窄夹缝、真空危险区
不能无视空间封锁、锚点破坏和位移扰流
携带物品、队友或敌人会显著增加消耗
连续瞬移会造成空间眩晕、方向感错乱和神经疲劳
长距离或高阶折叠需要更多定位信息和准备时间
换位类技能受目标质量、抗性、意志或空间稳定性影响
误判落点可能导致坠落、撞击、肢体错位或被困
```

能力者承载也必须限制：

```text
身体需要承受瞬间方向变化
大脑需要处理落点信息和空间坐标
内耳和平衡感容易被连续瞬移破坏
携带他人瞬移时需要承担对方姿态和质量带来的额外负荷
```

常见反制方式：

```text
空间封锁：禁止区域内生成落点
锚点破坏：清除或污染预设锚点
落点诱导：伪造安全落点引诱瞬移
位移扰流：让瞬移偏离原定位置
感知遮蔽：让能力者无法确认目标区域
重量压制：让换位或携带瞬移消耗暴涨
```

## 总结

瞬移能力的核心不是“想去哪就去哪”，而是围绕位置、落点、锚点和空间关系进行技能选择。

低阶瞬移解决“我怎么移动自己”，中阶瞬移解决“我怎么围绕锚点和目标改变位置”，高阶瞬移解决“我怎么改变局部空间距离”。玩家真正使用的是当前装配出的技能套组，而不是整棵技能树。

判断这套能力是否成立，重点看三件事：

```text
技能树是否能体现从短距位移到空间折叠的成长
关键技能是否能被装配成不同玩法流派
每个场景是否能说清技能如何使用、产生什么结果、付出什么代价
```
