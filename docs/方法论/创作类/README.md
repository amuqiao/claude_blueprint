# 创作类方法论

> 本目录存放用于构想、建模和完善虚构设定的方法论。

## 当前文档

- [`超能力构想与抽象升级方法论.md`](超能力构想与抽象升级方法论.md)
  用于把一个模糊能力种子，构想成有原生定义、能力范围、能力画像、应用场景、抽象升级和边界代价的自洽能力体系。

- [`超能力技能化与成长树方法论.md`](超能力技能化与成长树方法论.md)
  用于把一个能力想法锚点，转化为可分层、可点亮、可升级、可装备、可组合、可在场景中使用的技能体系。

- [`Dishonored风格能力组与升级路线方法论.md`](Dishonored风格能力组与升级路线方法论.md)
  用于参考 Dishonored 系列的沉浸式潜行动作玩法，从一个创意种子出发，设计一组可组合、可升级、可用于任务解法的能力。

如果目标是先形成能力本体设定，优先使用“超能力构想与抽象升级方法论”；如果目标是把能力转成可点亮、可升级、可装配的技能体系，优先使用“超能力技能化与成长树方法论”；如果目标是从一个创意种子设计类似 Dishonored 的能力组、升级路线和任务解法，优先使用“Dishonored 风格能力组与升级路线方法论”。

## 维护边界

创作类方法论负责帮助虚构设定形成可复用的构想和判断结构。

本目录不存放具体角色设定、剧情正文、世界观百科或单次脑洞记录；如果内容只是某一个能力的成品设定，应放入对应创作项目文档，而不是沉淀为方法论。
