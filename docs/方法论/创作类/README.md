# 创作类方法论

> 本目录存放用于构想、建模和完善虚构设定的方法论。

## 当前文档

- [`Dishonored风格能力组与升级路线方法论.md`](Dishonored风格能力组与升级路线方法论.md)
  用于参考 Dishonored 系列的沉浸式潜行动作玩法，从一个创意种子出发，设计一组可组合、可升级、可用于任务解法的能力。

如果目标是从一个创意种子设计类似 Dishonored 的能力组、升级路线和任务解法，优先使用“Dishonored 风格能力组与升级路线方法论”。

## 维护边界

创作类方法论负责帮助虚构设定形成可复用的构想和判断结构。

本目录不存放具体角色设定、剧情正文、世界观百科或单次脑洞记录；如果内容只是某一个能力的成品设定，应放入对应创作项目文档，而不是沉淀为方法论。
