# 方法论治理类

> 本目录存放用于新增、质检和维护方法论的方法论。

## 当前文档

- [`方法论新增范式.md`](方法论新增范式.md)
  用于判断一种思考方式是否值得升格为方法论，并说明新增方法论文档时应如何建立主干模型、节点和边界。

- [`方法论文档质检方法论.md`](方法论文档质检方法论.md)
  用于检查已有方法论文档是否对象清楚、命题成立、节点必要、表达得当、边界稳定。

## 维护边界

本目录不负责解释具体主题、具体需求或具体方案。

它负责维护“方法论如何形成”和“方法论如何被检查”的元规则，避免方法论文档退化成普通说明、流程清单、模板堆叠、无判断力的信息堆叠或临时经验。
